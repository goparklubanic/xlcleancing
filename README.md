# XLCLEANCING

Applikasi berbasis python untuk meng-ekstrak data invoice berformat ms excel.

## Requirements

- Python 3.x
- openpyxl
- formulas

## Installation

```bash
mkdir compressed kelar
pip install -r requirements.txt
```

## How to use

1. Jalankan app.py
2. Hasilnya ada di folder results/itemlist.xlsx
